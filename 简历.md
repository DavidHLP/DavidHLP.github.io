## icon:project 项目经历

**ResiCache 缓存防护中间件**

:::
项目地址：icon:github [https://github.com/DavidHLP/ResiCache](https://github.com/DavidHLP/ResiCache)

:::
**2025.01 - 至今**

::: end
**技术栈**：`Spring Boot 3` `Spring Cache` `Redis` `Redisson` `Caffeine` `责任链模式` `SPI` `Micrometer`
**项目描述**：ResiCache 是一个面向 Spring Boot 应用的 **缓存防护组件**，围绕缓存穿透、击穿、雪崩等问题，提供 **注解驱动的缓存治理能力**，减少业务代码中重复的缓存保护逻辑。
- **注解驱动缓存能力**：自研 `@RedisCacheable`、`@RedisCacheEvict`、`@RedisCaching` 等注解，基于 Spring AOP 拦截缓存调用流程，将 **缓存读取、空值处理、同步锁、TTL 扰动、预刷新** 等逻辑从业务代码中解耦。

- **责任链式防护设计**：设计 `CacheHandlerChain` 缓存处理链，将 BloomFilter、SyncLock、PreRefresh、TTL、NullValue、ActualPut 等策略拆分为独立 Handler，降低单个模块复杂度，便于后续扩展和替换。

- **缓存穿透防护**：实现 **JVM BitSet + Redis Hash Pipeline 的分层布隆过滤方案**，并结合空值缓存策略，对不存在的数据进行短期缓存，降低恶意请求或异常流量对数据库的冲击。

- **缓存击穿防护**：基于 **JVM 本地锁与 Redisson 分布式锁** 实现两级同步控制，避免热点 Key 失效后大量请求同时回源数据库。

- **缓存雪崩防护**：在基础 TTL 上引入 **随机扰动机制**，打散大量 Key 同时过期的问题；结合异步预刷新机制，对热点缓存进行提前更新，降低请求高峰期回源压力。

- **可观测与测试保障**：通过 Micrometer 暴露 **Hit、Miss、P95、P99** 等指标，并提供 HealthIndicator 健康检查；使用 Testcontainers 构建 Redis 集成测试，提升缓存逻辑验证可靠性。

**在线编程评测系统 UltiCode**

:::
项目地址：icon:github [https://github.com/DavidHLP/UltiCode](https://github.com/DavidHLP/UltiCode)

:::
**2025.10 - 至今**

::: end
**技术栈**：`Spring Boot 3` `Spring Security` `MyBatis-Plus` `Redis` `Redisson` `JWT` `Flyway` `MeiliSearch` `WebSocket/STOMP` `Docker` `Vue 3`
**项目描述**：UltiCode 是一个在线编程评测平台，支持 **题目练习、代码提交、沙箱判题、竞赛排名、用户权限、实时通知和后台管理** 等功能。项目采用前后端分离架构，后端围绕判题、竞赛、用户、权限、题库、搜索等模块进行拆分。
- **多语言判题沙箱**：基于 Docker 实现 Python、Java、C、C++ 代码隔离执行，通过 **non-root 用户、能力裁剪、seccomp 限制、资源限制** 等方式降低用户代码执行风险；支持编译错误、运行错误、超时、内存超限等结果分类。

- **异步判题任务流转**：使用 **Redis Stream / Consumer Group** 实现判题任务投递与消费，通过任务唯一标识和幂等去重机制避免重复判题；结合 worker 心跳和失败重试机制，提高判题任务处理稳定性。

- **实时竞赛与排行榜**：支持 ICPC、IOI、自定义赛制以及虚拟赛，设计提交记录、题目得分、罚时、最后通过时间等维度的排名规则；将原有排行榜计算逻辑从重复遍历优化为聚合计算，降低竞赛榜单刷新开销。

- **安全鉴权设计**：实现 **Access Token + Refresh Token 双 Token 登录机制**，Refresh Token 入库前进行哈希存储；结合 RBAC 权限模型、接口级权限校验、CSRF Token 校验和操作审计，降低越权访问和敏感操作风险。

- **缓存与搜索能力**：使用 **Spring Cache + Caffeine + Redis** 构建多级缓存，对题目、题解、用户信息等高频数据进行缓存；接入 MeiliSearch 支持题库、题解和帖子全文检索，并通过异步事件机制同步索引数据。

- **工程化与运维支持**：使用 Flyway 管理数据库迁移脚本，保证表结构变更可追踪；接入 Actuator、Micrometer、Prometheus 指标暴露，便于观察 Redis、判题 worker、搜索服务等关键组件状态。
